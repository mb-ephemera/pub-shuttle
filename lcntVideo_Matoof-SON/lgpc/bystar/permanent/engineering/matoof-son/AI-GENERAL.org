* General: Running LaTeX on Inputs

Make sure that lcntProc.sh has NOTYET

To build the document, cd into the relevant build directory (e.g. matoof-son-26/) and run:
lcntProc.sh -h -v -n showRun -p pre="clean" -p extent="build+view" -i lcntBuild cur
